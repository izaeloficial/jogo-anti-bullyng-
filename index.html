Como Jogar

Movimento

Mova-se entre as 3 pistas, pule trens e deslize por baixo de obstáculos.

← / APista esquerda
→ / DPista direita
↑ / WPular
↓ / SDeslizar

Poderes

  • 🛡️ Escudo — protege de uma colisão Espaço
  • ⚡ Velocidade — acelera e multiplica XP Q
  • 🧲 Ímã — atrai moedas e gemas E
  • 🌀 Lentidão — facilita os desvios R
  • 🚀 Jetpack — voe sobre as pistas F

No Celular

Deslize para os lados para trocar de pista, para cima para pular e para baixo para escorregar. Use os botões de poder na tela.

Dilemas

De tempos em tempos o jogo pausa e mostra uma situação real de bullying. Escolha a atitude mais empática para ganhar XP.
